iOS Photo Server

v1.3.1 latest

Serve your iPhone's photo library over the local network. REST API for agentic access, plus a browser UI for browsing.

⬇ Download Script

Setup

  1. Tap Download above
  2. Open the file in Pythonista 3
  3. Tap ▶ Play to start the server
  4. Grant photo access when prompted
  5. The server URL will appear in the console
